# Manual smoke checklist

> Чек-лист перед релизом. Прохождение — обязательно. Записать видео ≤ 5 мин в `docs/demos/v{version}-smoke.mp4`.

## Установка

- [ ] Скачан `Foliant-Setup-{version}-{tier}.exe` с GitHub Releases.
- [ ] SHA256 из `SHA256SUMS` совпадает.
- [ ] Установка на чистой Win10 21H2 VM — без warnings SmartScreen.
- [ ] Установка на чистой Win11 VM — без warnings SmartScreen.
- [ ] Иконка в Start, ярлык на рабочем столе (если выбрано).
- [ ] Приложение запускается, главное окно открывается за < 2 с (cold start).
- [ ] В `%LOCALAPPDATA%\Foliant\Logs\foliant-{date}.log` — запись «App started».

## Базовый просмотр (S1, S2)

- [ ] Открыт 10-страничный текстовый PDF.
- [ ] Скролл, PgUp / PgDn, Home / End — работает плавно.
- [ ] Zoom Ctrl+`+` / Ctrl+`-` / Ctrl+`0` — работает.
- [ ] Per-Monitor V2: перетаскивание окна между мониторами с разным DPI — без артефактов.
- [ ] Переключение темы Light / Dark / High Contrast — все страницы перерисовываются мгновенно.

## Кэш (S3, S4)

- [ ] Закрыть документ, открыть снова: страница появляется ≤ 20 мс (warm cache).
- [ ] Удалить документ, открыть тот же путь — кэш подхватывает.
- [ ] Изменить файл (touch), открыть — кэш инвалидирован.
- [ ] Settings → Очистить кэш — каталог `%LOCALAPPDATA%\Foliant\Cache\` пуст.

## Локализация и Settings (S5)

- [ ] Settings → Language → English — UI переключается без перезапуска.
- [ ] Settings → Language → Русский — обратно.
- [ ] Recents показывает последние 20 открытых файлов.
- [ ] Recents → правый клик → «Удалить и кэш» — файл и его кэш удалены.

## Поиск (S6, S7)

- [ ] Ctrl+F → ввод слова → найдено N результатов с подсветкой.
- [ ] F3 / Shift+F3 — навигация по результатам.
- [ ] На 500-страничном документе поиск ≤ 1 с холодно, ≤ 100 мс тепло.
- [ ] FTS-индекс: открыть второй документ, искать — индекс обоих.

## OCR + DjVu (S8, S9)

- [ ] Открыт 10-страничный скан.
- [ ] OCR → язык Русский → запуск → прогресс по страницам → завершение ≤ 30 с.
- [ ] Поиск по результату работает.
- [ ] DjVu plugin установлен.
- [ ] Открыт DjVu — отрендерено, текстовый слой доступен (если был).

## Аннотации и страницы (S10, S11)

- [ ] Highlight → закрыть → открыть — на месте.
- [ ] Sticky note → текст → сохранить → перезапуск приложения — на месте.
- [ ] Freehand → нарисовать → сохранить.
- [ ] Удалить страницу 5 из 10-страничного PDF, сохранить, открыть в Acrobat / Foxit — корректная структура.
- [ ] Reorder страниц через thumb-strip — drag works, save works.

## Редактор + конвертер (S12)

- [ ] Заменить одну строку текста, сохранить, открыть в другом просмотрщике — изменено.
- [ ] Экспорт в DOCX — открывается в Word, текст распознан.

## License + триал (S13)

- [ ] Чистый запуск: триал активен 30 дней.
- [ ] Открыт Pro-feature → нет блокировок.
- [ ] Перевод системного времени на +35 дней → перезапуск → триал истёк.
- [ ] Перевод обратно → триал по-прежнему истёк (ratchet).
- [ ] Активация лицензией: ввод ключа → принят → Pro-feature работает.
- [ ] Проверка, что `%APPDATA%\Foliant\license.key` — DPAPI (открытие из другого пользователя — неудача).

## Uninstall

- [ ] Панель управления → Программы → Foliant → Удалить.
- [ ] `%ProgramFiles%\Foliant\` — удалён.
- [ ] `%LOCALAPPDATA%\Foliant\` — оставлен (опция «удалить мои данные» во встроенном диалоге uninstaller — спрашивается).
- [ ] `%APPDATA%\Foliant\` — то же поведение.
- [ ] Reg key `HKCU\Software\Foliant\` — оставлен / удалён согласно выбору.

## Регрессии

- [ ] Performance baseline pass (см. `tests/Foliant.Performance/baseline.json`).
- [ ] Лог за всю сессию smoke — нет уровня Error / Fatal.
- [ ] CrashReports — пуст.
